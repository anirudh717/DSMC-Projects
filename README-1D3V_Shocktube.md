1) The 1D3V_Shocktube.py is a One-dimensional Three-velocity shock tube simulator using DSMC.
2) The code has been adapted from Dr. Garcia's DSMC MATLAB code:
Alejandro Garcia (2025). Numerical Methods for Physics, 2e 
(https://www.mathworks.com/matlabcentral/fileexchange/2270-numerical-methods-for-physics-2e), MATLAB Central File Exchange. Retrieved December 20, 2025.
3) This code is the initial version, which will be updated to a more robust and versatile DSMC solver soon.
